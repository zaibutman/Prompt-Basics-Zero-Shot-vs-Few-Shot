# Project Requirements

## Overview

This project evaluates the effectiveness of **Zero-Shot Prompting** and **Few-Shot Prompting** using Large Language Models (LLMs) for customer support message classification.

Since this is a prompt engineering assignment, no programming libraries or external dependencies are required.

---

# Software Requirements

- Modern Web Browser (Chrome, Edge, Firefox, etc.)
- Internet Connection
- Git (optional, for version control)
- GitHub Account (recommended)

---

The experiment can be performed using any of the following AI models:
